# ☁️ Static Website Hosting on AWS

## 📌 Project Overview
This project demonstrates how a simple static website can be hosted on AWS using a scalable and cost-effective architecture.

The website was built using HTML and CSS and designed to be deployed on AWS cloud services.

---

## 🛠️ Technologies Used
- HTML
- CSS

---

## ☁️ AWS Services Used
- Amazon S3 (Static Website Hosting)
- Amazon CloudFront (Content Delivery Network)
- Route 53 (Domain Name System)

---

## 🧱 Architecture
The website files are stored in Amazon S3 and delivered to users through CloudFront for improved performance and low latency.

User → CloudFront → S3

---

## 🔐 Security
- HTTPS enabled using SSL/TLS
- Access control via IAM policies
- Secure content delivery

---

## ⚙️ Deployment Steps
1. Create an S3 bucket
2. Upload website files
3. Enable static website hosting
4. Configure CloudFront distribution
5. Connect CloudFront to S3

---

## 🚀 Outcome
This project demonstrates foundational knowledge of cloud computing and AWS architecture, aligned with the AWS Cloud Practitioner certification.
